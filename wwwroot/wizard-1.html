Welcome to the Platform developer runtime

This wizard will walk you through the steps required to configure (or reconfigure) the Cloud Services Platform on your machine.

System Requirements:

  • Windows 8.x (the developer runtime may work on Windows 7, but it's not officially supported).
  • Powershell 4 or higher (this is required by the Aperture Powershell module).

Please ensure that you have already installed:

  • RavenDB 3.x
  • Seq
  • Semantic Logging Service
